## About Me

Versatile full-stack programmer with core strength in Python, Django, and React, and the ability to drop into C++ when performance matters. I build backend systems, APIs, and data-driven applications, and care about clean architecture that's still readable six months later.

- Backend: Python, Django, REST APIs, PostgreSQL
- Frontend: React, TypeScript
- Systems: C++ for performance-critical work
- Infra: Docker, AWS
