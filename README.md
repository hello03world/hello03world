## About Me

Versatile full-stack programmer who builds, ships, **and** runs what I build. Core strength in Python, Django, and React, with the ability to drop into C++ when performance matters — and the DevOps chops to take an app from `localhost` to a production server that stays up.

```yaml
name:        Hello World
role:        Versatile Programmer — Full-Stack & DevOps
focus:       Backend systems, APIs, data-driven apps, deployment
strengths:   Clean architecture that's still readable six months later
currently:   Building backends and owning the path to production
mindset:     "If I wrote it, I can deploy it, scale it, and keep it alive."
```

## What I Do

🔹 **Backend Engineering** — REST APIs, business logic, auth, background jobs, and database design with Python & Django.

🔹 **Frontend** — Responsive, component-driven UIs with React & TypeScript.

🔹 **Deployment & Servers** — I take projects to production: Linux servers, Nginx/Gunicorn, Docker, environment config, SSL, and zero-downtime releases.

🔹 **DevOps & Infra** — CI/CD pipelines, containerization, cloud (AWS), and keeping services monitored and reliable.

🔹 **Systems / Performance** — C++ for the performance-critical paths where it counts.
